# GitHub-Page
Diretório em que tenho os documentos a serem inseridos no GitHub Pages.
Este repositório foi criado para armazenar e documentar o desenvolvimento do meu Site Portfólio, um projeto prático desenvolvido ao longo da disciplina Bootcamp I no UniCEUB.
O principal objetivo deste projeto é aplicar os conceitos fundamentais de desenvolvimento web, criando uma vitrine virtual que centralize minhas competências técnicas, experiências profissionais, projetos acadêmicos e formas de contato.
